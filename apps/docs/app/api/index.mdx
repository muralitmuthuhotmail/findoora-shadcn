# API Reference

Welcome to the Findoora API documentation. Our REST API allows you to integrate Findoora's powerful search and content management capabilities into your applications.

## Base URL

```
https://api.findoora.com/v1
```

All API requests require authentication using an API key:

```http
Authorization: Bearer YOUR_API_KEY
```

Learn more about [authentication methods](/api/authentication).

## Core Endpoints

### Search

Search through your content and get relevant results.

```http
GET /search?q={query}
```

**Example:**
```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     "https://api.findoora.com/v1/search?q=documentation&limit=10"
```

### Content Management

Manage your content programmatically.

```http
GET    /content           # List content
POST   /content           # Create content
GET    /content/{id}      # Get specific content
PUT    /content/{id}      # Update content
DELETE /content/{id}      # Delete content
```

### Collections

Organize content into collections.

```http
GET    /collections       # List collections
POST   /collections       # Create collection
GET    /collections/{id}  # Get collection
PUT    /collections/{id}  # Update collection
DELETE /collections/{id}  # Delete collection
```

## Response Format

All API responses follow a consistent JSON format:

```json
{
  "success": true,
  "data": {
    // Response data
  },
  "meta": {
    "page": 1,
    "per_page": 20,
    "total": 100,
    "total_pages": 5
  }
}
```

### Error Responses

Error responses include detailed information:

```json
{
  "success": false,
  "error": {
    "code": "INVALID_QUERY",
    "message": "The search query is invalid",
    "details": {
      "field": "q",
      "reason": "Query cannot be empty"
    }
  }
}
```

## Rate Limits

The API enforces rate limits to ensure fair usage:

- **Free tier**: 1,000 requests per hour
- **Pro tier**: 10,000 requests per hour
- **Enterprise**: Custom limits

Headers in each response show your current usage:

```http
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1609459200
```

Learn more about [rate limits and best practices](/api/rate-limits).

## Examples and Tutorials

### Basic Search Integration

```javascript
// Search and display results
async function searchContent(query) {
  const response = await fetch(`https://api.findoora.com/v1/search?q=${query}`, {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  
  const data = await response.json();
  return data.results;
}
```

### Creating Content

```javascript
// Add new content to Findoora
async function addContent(title, content, tags) {
  const response = await fetch('https://api.findoora.com/v1/content', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title,
      content,
      tags
    })
  });
  
  return await response.json();
}
```
