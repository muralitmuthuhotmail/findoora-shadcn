## Quick Reference

### Essential Actions

| Action | Method |
|--------|---------|
| Search | Type in the search bar or press `Ctrl/Cmd + K` |
| Save item | Click the star icon or press `S` |
| Create collection | Go to Collections → New Collection |
| Share content | Click share icon and choose method |
| Access settings | Click profile menu → Settings |

### Common Workflows

**Finding and Saving Content:**
1. Enter search terms
2. Review results and apply filters if needed
3. Click on relevant items to view details
4. Save important items to collections or tag them

**Organizing Your Findings:**
1. Create collections for different topics
2. Add tags to items for flexible categorization
3. Use folders for hierarchical organization
4. Set up automated rules for consistent organization

**Collaborating with Team:**
1. Create a shared workspace
2. Invite team members with appropriate permissions
3. Share relevant collections and content
4. Use comments and annotations for discussions
