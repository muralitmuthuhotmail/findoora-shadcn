# Troubleshooting

This guide helps you resolve common issues you might encounter while using Findoora.

## Before You Start

Before diving into specific issues, try these general troubleshooting steps:

## Login & Authentication Issues

### Cannot Log In

**Symptoms:** Login page shows error messages or doesn't respond

**Solutions:**

import { Steps } from 'nextra/components'

<Steps>
### Verify your credentials

- Double-check your email address and password
- Make sure Caps Lock is off
- Try typing your password in a text editor first to verify it

### Reset your password

1. Click "Forgot Password" on the login page
2. Enter your email address
3. Check your inbox (and spam folder)
4. Follow the reset link

### Clear browser data

1. Clear cookies and cache for findoora.com
2. Disable browser extensions
3. Try incognito/private browsing mode

### Try a different browser

Test login with Chrome, Firefox, Safari, or Edge
</Steps>

### Two-Factor Authentication Issues

**Problem:** 2FA codes not working

**Solutions:**
- Ensure your device's time is synchronized
- Try generating a new code (codes expire after 30 seconds)
- Use backup codes if available
- Contact support to reset 2FA

## Search Problems

### No Search Results

**Symptoms:** Search returns empty results or "No results found"

**Troubleshooting:**

| Issue | Solution |
|-------|----------|
| **Typos in query** | Check spelling, try variations |
| **Too specific search** | Use broader, simpler terms |
| **Filters too restrictive** | Clear filters and try again |
| **Content not indexed yet** | Wait a few minutes for new content |
| **Insufficient permissions** | Check if you have access to the content |

### Slow Search Performance

**Symptoms:** Search takes a long time to return results

**Solutions:**
- Check your internet connection speed
- Try simpler search queries
- Clear browser cache
- Close unnecessary browser tabs
- Contact support if consistently slow

### Irrelevant Search Results

**Problem:** Search results don't match your query

**Improvements:**
- Use exact phrases in quotes: `"project management"`
- Add negative terms: `cats -dogs`
- Use specific filters: `type:pdf`, `date:2024`
- Try different keyword combinations

## Content Management Issues

### Cannot Save Content

**Symptoms:** Save button doesn't work or content disappears

**Diagnostics:**

```
Error Types:
├── Network Issues
│   ├── Check internet connection
│   ├── Try refreshing the page
│   └── Wait and retry
├── Storage Limits
│   ├── Check account storage usage
│   ├── Delete old content
│   └── Upgrade plan if needed
└── Permissions
    ├── Verify write access
    ├── Check workspace permissions
    └── Contact workspace admin
```

### Content Not Syncing

**Problem:** Content appears on one device but not others

**Resolution Steps:**

1. **Check sync status**
   - Look for sync indicator in the app
   - Wait for sync to complete

2. **Force sync**
   - Pull down to refresh (mobile)
   - Press Ctrl/Cmd + R (desktop)

3. **Verify account**
   - Ensure same account on all devices
   - Log out and log back in

4. **Check connectivity**
   - Ensure stable internet connection
   - Try switching networks

### Missing Content

**Symptoms:** Previously saved content has disappeared

**Recovery options:**
- Check the Trash/Deleted Items folder
- Look in different collections or folders
- Use search to locate moved content
- Check if content was shared and permissions changed
- Contact support for data recovery

### Team Members Cannot Access Shared Content

**Symptoms:** Invited users report they cannot see shared content

**Solutions:**
1. **Verify invitations**
   - Check if invitations were sent successfully
   - Ask team members to check spam folders
   - Resend invitations if needed

2. **Check permissions**
   - Ensure correct permission levels
   - Verify workspace access
   - Update permissions if needed

3. **Account status**
   - Confirm team members have active accounts
   - Check if accounts are verified
   - Ensure they're in the correct workspace

## Performance Issues

### App Loading Slowly

**Symptoms:** Findoora takes a long time to load or respond

**Performance optimization:**

```
Optimization Checklist:
□ Clear browser cache and cookies
□ Close unnecessary browser tabs
□ Disable heavy browser extensions
□ Check available RAM and CPU
□ Try a different browser
□ Test internet speed
□ Restart browser/device
```

### High Memory Usage

**Problem:** Browser uses too much memory with Findoora

**Memory management:**
- Close unused tabs and windows
- Clear browser cache regularly
- Limit number of open collections
- Use browser task manager to identify issues
- Consider using desktop app instead

## Mobile App Issues

### App Crashes

**Symptoms:** Mobile app closes unexpectedly

**Solutions:**
1. **Update the app**
   - Check App Store/Google Play for updates
   - Install latest version

2. **Restart the app**
   - Force close and reopen
   - Restart your device

3. **Clear app data**
   - Clear cache (Android)
   - Offload and reinstall (iOS)

4. **Check device resources**
   - Ensure sufficient storage space
   - Close other apps to free memory

### Sync Issues on Mobile

**Problem:** Mobile app not syncing with web version

**Resolution:**
- Check mobile data/WiFi connection
- Enable background app refresh
- Log out and log back in
- Update to latest app version
- Contact support if issues persist

## Integration Problems

### Third-Party Integration Fails

**Symptoms:** Connected services stop working or show errors

**Troubleshooting:**

| Service | Common Issues | Solutions |
|---------|--------------|-----------|
| **Google Drive** | Permission expired | Reconnect in settings |
| **Slack** | Bot not responding | Re-add Findoora bot |
| **GitHub** | Repos not syncing | Check repository permissions |
| **Email** | Not receiving notifications | Check spam folder, update email settings |

### API Connection Issues

**Problem:** API requests failing or timing out

**Debugging steps:**
1. **Check API key**
   - Verify key is correct and active
   - Generate new key if needed

2. **Test connectivity**
   - Use curl or Postman to test endpoints
   - Check for network firewalls

3. **Review rate limits**
   - Check if you've exceeded limits
   - Implement proper rate limiting

4. **Examine error responses**
   - Look at HTTP status codes
   - Check error messages for details

## Browser-Specific Issues

### Chrome Issues

**Common problems:**
- Extensions blocking functionality
- Outdated browser version
- Corrupted profile data

**Solutions:**
- Try incognito mode
- Disable extensions
- Update Chrome
- Create new browser profile

### Safari Issues

**Known limitations:**
- Some features require specific Safari versions
- Cookie settings may block functionality

**Fixes:**
- Update to latest Safari
- Check privacy settings
- Allow cookies for findoora.com

### Firefox Issues

**Potential problems:**
- Strict privacy settings
- Add-ons interfering

**Resolutions:**
- Adjust privacy settings
- Disable add-ons temporarily
- Clear Firefox cache

## Error Messages

### Common Error Codes

| Error Code | Meaning | Solution |
|------------|---------|----------|
| **401** | Unauthorized | Log in again or check API key |
| **403** | Forbidden | Check permissions or contact admin |
| **404** | Not Found | Verify content exists and you have access |
| **429** | Rate Limited | Wait and retry, or upgrade plan |
| **500** | Server Error | Try again later or contact support |

### Network Errors

**"Connection Failed" or "Network Error"**

1. Check internet connection
2. Try different network (mobile data vs WiFi)
3. Disable VPN temporarily
4. Check firewall settings
5. Contact IT department if on corporate network

Gather this information:

- Operating system
- Account email address
- Steps to reproduce the issue
- Error messages (screenshots helpful)
- Network environment (home, office, etc.)
